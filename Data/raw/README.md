# Instructions for raw datasets
There are three raw datasets required: 
* `credit_card_transactions-ibm_v2.csv`
* `sd254_cards.csv`
* `sd254_users.csv`

`credit_card_transactions-ibm_v2.csv` is too big to be uploaded to GitHub. Please download the file from [Kaggle Credit Card Transactions](https://www.kaggle.com/datasets/ealtman2019/credit-card-transactions/data)
